ΟΔΗΓΙΕΣ ΓΙΑ ΤΟ ΠΡΟΓΡΑΜΜΑ
1)Ανοίγουμε το cmd και εγκαθιστούμε την mysql-connector .
>>pip install mysql-connector
2)Εγκαθιστούμε την pyqt5
>>pip install pyqt5
>>pip install pyqt5-tools
3)Στον φάκελο με τον κώδικα πρέπει να συμπεριλάβετε και την φωτογραφία που υπάρχει στο συμπιεσμένο αρχείο για να την αναγνωρίζει ο κώδικας.



ΧΡΗΣΗ ΕΦΑΡΜΟΓΗΣ
Το κύριο πρόγραμμα βρίσκεται στο Menu.py και αφού το τρέξετε θα σας εμφανιστούν όλες οι διαθέσιμες λειτουργίες ( Πελάτες, Παραστάσεις , Διαθέσιμα Εισητήρια , Κράτηση)

Σημείωση:
Για να πατηθεί το κουμπί Plithos και να εμφανιστεί το πλήθος των διαθέσιμων εισιτηρίων πρέπει να τρέξετε το DiathesimaEishthria.py (το αποτέλεσμα θα εμφανιστεί στο terminal)

Για να γίνει η κράτηση πρέπει:
1.Να τρέξετε το Menu.py και να επιλέξετε έναν κωδικό από τα διαθέσιμα εισιτήρια
2.Να τρέξετε το insert.py και προσθέστε τα στοιχεία του πελάτη και τον κωδικό του επιλεγμένου εισιτηρίου
3.Ξανατρέξτε το Menu.py και πατώντας την λίστα των πελατών θα έχει προστεθεί ο νέος πελάτης και πατώντας την λίστα των διαθέσιμων εισιτηρίων θα έχει διαγραφεί το αντίστοιχο εισιτήριο
